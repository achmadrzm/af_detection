## Installation

### 1. Clone repository
```bash
git clone https://github.com/achmadrzm/af_detection.git
```

```bash
cd af_detection
```

### 2. Create virtual environment
```bash
python -m venv af
```

### 3. Activate virtual environment
#### Windows:
```bash
af\Scripts\activate
```

#### Linux/MacOS:
```bash
source af/bin/activate
```

### 4. Install library
```bash
pip install -r requirements.txt
```
